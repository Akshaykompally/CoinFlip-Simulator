# 🪙 CoinFlip Simulator

A simple and interactive web application that simulates a real coin toss using **HTML**, **CSS**, and **JavaScript**. Click the **Flip Coin** button to generate a random **Heads** or **Tails** result with visual feedback.

---

## 📖 Project Overview

**CoinFlip Simulator** is a beginner-friendly JavaScript project that demonstrates the use of random number generation, DOM manipulation, and event handling to simulate flipping a coin.

The application randomly displays either **Heads** or **Tails** using images, making it a fun and engaging way to learn front-end web development.

---

## ✨ Features

- 🪙 Random coin toss simulation
- 🎲 Equal probability for Heads and Tails
- 🖼️ Displays corresponding coin image
- ⚡ Fast and responsive interface
- 📱 Lightweight web application
- 💻 Beginner-friendly project

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Akshaykompally/Flip-A-Coin.git
```

### Open the Project

Simply open the **index.html** file in your preferred web browser.

No installation or dependencies are required.

---

## 🎮 How to Use

1. Open the application.
2. Click the **Flip Coin** button.
3. The application randomly selects:

- 🪙 Heads
- 🪙 Tails

4. The corresponding image is displayed instantly.
